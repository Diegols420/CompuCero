# CompuCero | E-Commerce de Hardware & Componentes 🚀

[![Estado del Proyecto](https://img.shields.io/badge/Status-En%20Desarrollo-orange)](https://github.com/)
[![Tecnologías](https://img.shields.io/badge/Stack-Frontend%20Vanilla-blue)](https://github.com/)

**CompuCero** es una plataforma web de comercio electrónico enfocada en la venta de hardware de alta gama y componentes de PC. El proyecto nace bajo un enfoque académico en la **Universidad Tecnológica del Perú (UTP)**, simulando un entorno de negocio real que exige una experiencia de usuario (UX) fluida, un diseño visual inmersivo (Estilo Dark/Gamer) y un rendimiento optimizado.

---

## 🎯 Objetivos del Proyecto & Enfoque Técnico
Este repositorio fue creado con el fin de demostrar habilidades sólidas en el desarrollo Frontend utilizando tecnologías nativas (Vanilla Web Development). Los principales focos de atención durante el desarrollo han sido:
* **Semántica Estricta:** Uso correcto de etiquetas HTML5 para garantizar accesibilidad y SEO básico.
* **Arquitectura de Estilos:** Organización limpia de reglas CSS3, priorizando la escalabilidad y el mantenimiento del diseño visual.
* **Componentización:** Estructura modular de carpetas para facilitar la migración futura a Frameworks como React o Vue.

---

## 💻 Características Implementadas
* **Landing Page Dinámica:** Presentación de productos destacados (ej. stock de GPUs de última generación) con llamadas a la acción (CTA) claras.
* **Sección Corporativa ("Nosotros"):** Vista optimizada para la narrativa de la marca, visión, misión y métricas del equipo del proyecto.
* **Módulo de Carrito de Compras:** Interfaz intuitiva orientada a la conversión y experiencia de usuario limpia.
* **Diseño UI Responsivo:** Adaptabilidad para dispositivos móviles, tablets y pantallas de escritorio.

---

## 🛠️ Stack Tecnológico
* **HTML5** - Estructuración semántica del contenido.
* **CSS3** - Estilos personalizados, Flexbox/Grid Layouts, y variables nativas para la paleta de colores.

```text
